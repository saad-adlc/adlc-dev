# adlc-dev

Agent workspace repository for the Orix ADLC pipeline.
All code is generated by Claude Code agents following `adlc-standards`.

## Structure

```
workspaces/
  issue-<N>-<slug>/   ← one self-contained workspace per issue
    src/
    package.json
    .gitignore
    spec.md
    vite.config.ts / tsconfig.json / etc.
```

## Rules

- Never push directly to `main` — all changes via feature branches and PRs
- Every workspace is fully self-contained — no shared `node_modules`
- PRs from feature branches target `adlc-staging/apps/<app-name>`
- `node_modules/`, `dist/`, `coverage/` are blocked globally by root `.gitignore`

## Branching convention

Feature branches: `feature/issue-<N>-<short-description>`
Always branch from `main`. Never from another feature branch.
